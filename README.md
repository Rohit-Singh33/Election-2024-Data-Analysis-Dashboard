## Overview
This Power BI dashboard visualizes the **2024 Indian General Election Results** with clear insights into national and state-level outcomes.  
It provides an interactive breakdown of seats by alliance, party, and candidate.


##  Features
- National-level summary of seats (NDA, I.N.D.I.A, and Others)
- State-wise detailed results with charts and filters
- Party-wise and alliance-wise breakdown
- Dynamic visuals for easier comparison

- ##  Tools Used
- **Power BI Desktop**
- **Excel / CSV Data Source**
- **DAX Measures** for calculations
- **Custom Visuals** for pie and card charts
